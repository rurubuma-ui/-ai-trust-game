# 🧪 Как протестировать Telegram версию

## ✅ Текущий статус

Серверы запущены:
- ✅ Backend: `http://localhost:3000`
- ✅ Client: `http://localhost:4173`

```
http://localhost:4173
```

Игра должна работать, но функции Telegram не будут доступны.

## 🚀 Вариант 2: Полный тест через Telegram

### Шаг 1: Установите ngrok

**Способ 1: Через сайт**
1. Перейдите на https://ngrok.com/download
2. Скачайте для Windows
3. Распакуйте в удобную папку (например: `C:\ngrok`)

**Способ 2: Через Chocolatey (если установлен)**
```powershell
choco install ngrok
```

**Способ 3: Через PowerShell (как администратор)**
```powershell
# Скачать и установить автоматически
iwr https://bin.equinox.io/c/bNyj1mQVY4c/ngrok-v3-stable-windows-amd64.zip -OutFile ngrok.zip
Expand-Archive ngrok.zip -DestinationPath .
```

### Шаг 2: Запустите ngrok

Откройте новый терминал и выполните:

```bash
ngrok http 4173
```

Вы увидите что-то вроде:
```
Session Status                online
Account                       Your Account
Version                       3.x.x
Region                        United States (us)
Forwarding                    https://abc123def456.ngrok-free.app -> http://localhost:4173
```

**ВАЖНО:** Скопируйте HTTPS URL (например: `https://abc123def456.ngrok-free.app`)

### Шаг 3: Создайте бота через BotFather

1. Откройте Telegram
2. Найдите бота [@BotFather](https://t.me/botfather)
3. Отправьте команду `/newbot`
4. Следуйте инструкциям:
   - Введите имя бота (например: `Игра Нейросеть или реальность`)
   - Введите username бота (должен заканчиваться на `bot`, например: `neuronset_game_bot`)
5. **Сохраните токен бота** - он понадобится позже
   - Выглядит примерно так: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`

### Шаг 4: Создайте Web App

1. В чате с BotFather отправьте `/newapp`
2. Выберите вашего бота (из списка)
3. Заполните данные:

   **Title:** 
   ```
   Нейросеть или реальность?
   ```

   **Description:**
   ```
   Угадай, реальное фото или AI? Интересная игра на угадывание изображений.
   ```

   **Photo:** (можно пропустить, нажав `/empty`)

   **Web App URL:**
   ```
   https://ВАШ_NGROK_URL/client/
   ```
   
   Например, если ngrok дал `https://abc123def456.ngrok-free.app`, то URL будет:
   ```
   https://abc123def456.ngrok-free.app/client/
   ```
   
   **⚠️ ВАЖНО:** URL должен заканчиваться на `/client/`

4. Готово! BotFather подтвердит создание Web App

### Шаг 5: Протестируйте!

1. Откройте вашего бота в Telegram
2. Отправьте команду `/start`
3. Должна появиться кнопка с игрой (обычно внизу или как inline-кнопка)
4. Нажмите на кнопку
5. Игра откроется в Telegram Web App! 🎉

## 🐛 Решение проблем

### Проблема: Web App не открывается

**Решение:**
1. Проверьте, что ngrok запущен и работает
2. Убедитесь, что URL в BotFather правильный (заканчивается на `/client/`)
3. Проверьте, что клиент запущен на порту 4173
4. Попробуйте открыть URL напрямую в браузере - должна загрузиться игра

### Проблема: Ошибки CORS

**Решение:**
1. Убедитесь, что backend сервер запущен
2. Проверьте консоль браузера (DevTools) на ошибки
3. Убедитесь, что backend разрешает запросы от Telegram доменов

### Проблема: Данные не сохраняются

**Решение:**
- CloudStorage работает только в официальных клиентах Telegram
- В других клиентах используется localStorage (fallback)
- Проверьте консоль на ошибки сохранения

### Проблема: Имя пользователя не берется из Telegram

**Решение:**
- Убедитесь, что игра открыта через Telegram Web App (не просто в браузере)
- Проверьте, что `window.Telegram.WebApp` доступен
- Откройте DevTools и проверьте консоль

## 📝 Открытие DevTools в Telegram

**Telegram Desktop:**
- Нажмите `Ctrl+Shift+I` (Windows/Linux) или `Cmd+Option+I` (Mac)
- Или: Меню → Settings → Advanced → Enable Debug Mode

**Telegram Web:**
- Используйте DevTools браузера (F12)
