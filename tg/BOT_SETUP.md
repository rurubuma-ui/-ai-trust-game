# 🤖 Настройка Telegram бота

## ✅ Токен бота настроен!

Токен бота уже сохранен в `.env` файле.

## 🚀 Запуск бота

### Вариант 1: Отдельно

```bash
npm run bot
```

### Вариант 2: Все вместе (сервер + клиент + бот)

```bash
npm run dev:all
```

## ⚠️ ВАЖНО: Настройте Web App URL!

Перед запуском нужно:

1. **Запустить ngrok** (для разработки):
   ```bash
   ngrok http 4173
   ```

2. **Скопировать HTTPS URL** из ngrok (например: `https://abc123.ngrok-free.app`)

3. **Обновить `.env` файл:**
   ```env
   WEB_APP_URL=https://abc123.ngrok-free.app/client/
   ```
   
   Или обновить URL в BotFather:
   - Отправьте `/newapp` в BotFather
   - Выберите вашего бота
   - Обновите Web App URL

4. **Перезапустить бота**

## 📝 Команды бота

- `/start` - Начать игру (показывает кнопку Web App)
- `/help` - Показать справку

## 🔍 Проверка работы

1. Запустите бота: `npm run bot`
2. Откройте Telegram
3. Найдите вашего бота (по username, который вы указали при создании)
4. Отправьте `/start`
5. Должна появиться кнопка "🎮 Играть"
6. Нажмите на кнопку - игра должна открыться!

## 🐛 Решение проблем

### Бот не отвечает

- Проверьте, что токен правильный
- Убедитесь, что бот запущен (`npm run bot`)
- Проверьте консоль на ошибки

### Web App не открывается

- Убедитесь, что ngrok запущен
- Проверьте, что `WEB_APP_URL` в `.env` правильный
- URL должен заканчиваться на `/client/`
- Проверьте, что клиент запущен на порту 4173

### Ошибки в консоли

- Проверьте логи бота
- Убедитесь, что все зависимости установлены (`npm install`)
