# 🧪 Тестирование Telegram версии

## 📋 Пошаговая инструкция

### Шаг 1: Установка зависимостей

```powershell
cd tg
npm install
```

### Шаг 2: Запуск серверов

**В первом терминале (Backend):**
```powershell
cd tg
npm run dev:server
```

**Во втором терминале (Клиент):**
```powershell
cd tg
npm run dev:client
```

Или оба сразу:
```powershell
cd tg
npm run dev
```

### Шаг 3: Настройка ngrok (для доступа извне)

1. Установите ngrok: https://ngrok.com/download
2. Или через chocolatey: `choco install ngrok`
3. Запустите ngrok:
   ```bash
   ngrok http 4173
   ```
4. Скопируйте HTTPS URL (например: `https://abc123.ngrok.io`)

### Шаг 4: Создание бота через BotFather

1. Откройте Telegram
2. Найдите [@BotFather](https://t.me/botfather)
3. Отправьте `/newbot`
4. Следуйте инструкциям:
   - Имя: `Игра Нейросеть или реальность`
   - Username: `your_game_bot` (должен заканчиваться на `bot`)
5. Сохраните токен бота (понадобится позже)

### Шаг 5: Создание Web App

1. В BotFather отправьте `/newapp`
2. Выберите вашего бота
3. Заполните данные:
   - **Title**: Нейросеть или реальность?
   - **Description**: Угадай, реальное фото или AI?
   - **Photo**: Можно пропустить или загрузить позже
   - **Web App URL**: `https://ВАШ_NGROK_URL.ngrok.io/client/`
     Например: `https://abc123.ngrok.io/client/`

### Шаг 6: Тестирование

1. Откройте вашего бота в Telegram
2. Отправьте `/start`
3. Нажмите кнопку с игрой
4. Игра должна открыться в Telegram Web App

## 🔧 Быстрый тест без бота (локально)

Можно протестировать локально, открыв в браузере:
```
http://localhost:4173
```

Но для полноценного тестирования Telegram функций нужен реальный бот.

## ⚠️ Важно

- ngrok URL меняется при каждом перезапуске (в бесплатной версии)
- Для production используйте постоянный домен
- Убедитесь, что серверы запущены перед настройкой Web App

## 🐛 Решение проблем

### Web App не открывается
- Проверьте, что ngrok запущен
- Проверьте, что клиент запущен на порту 4173
- Убедитесь, что URL в BotFather правильный (должен заканчиваться на `/client/`)

### Ошибки CORS
- Проверьте, что backend запущен
- Убедитесь, что CORS настроен для Telegram доменов

### Данные не сохраняются
- CloudStorage работает только в официальных клиентах Telegram
- В других клиентах используется localStorage (fallback)
