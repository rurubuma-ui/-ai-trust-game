Источники изображений

Изображения для игры загружаются автоматически из следующих открытых источников:

  • Pexels - бесплатные стоковые фотографии высокого качества
  • Pollinations AI - AI-генерированные изображения с параметром nologo=true
  • LoremFlickr - случайные изображения по категориям (природа, города, люди)

Все изображения используются в соответствии с лицензиями источников. Разрешение изображений оптимизировано для быстрой загрузки (600x450px).

Правила игры

  • Имена пользователей должны соответствовать правилам сообщества
  • Запрещено использование оскорбительных, нецензурных или дискриминационных ников
