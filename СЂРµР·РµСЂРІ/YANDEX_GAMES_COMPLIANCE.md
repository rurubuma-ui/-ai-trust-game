## ⚠️ Проблемы, которые нужно исправить:

### 1. **КРИТИЧНО: Использование `window.prompt` и `window.alert`**

**Проблема:** Яндекс Игры не разрешают использование нативных браузерных диалогов.

**Места:**
- `client/main.js:1735` - `window.prompt` для изменения имени
- `client/main.js:1739` - `window.alert` для ошибок
- `client/main.js:2363` - `window.alert` для ошибок WebSocket

**Решение:** Заменить на кастомные модальные окна.

### 2. **Рекомендуется: Интеграция рекламы**

**Проблема:** Нет интеграции с рекламным SDK Яндекс Игр.

**Решение:** Добавить:
- `showFullscreenAdv()` - полноэкранная реклама
- `showRewardedVideo()` - реклама за награды (опционально)
- `showBannerAdv()` - баннерная реклама (опционально)

### 3. **Рекомендуется: Интеграция лидербордов**

**Проблема:** Лидерборды не интегрированы с SDK.

**Решение:** Использовать `ysdk.getLeaderboards()` для глобальных таблиц лидеров.

### 4. **Внешние источники изображений**

**Проблема:** Используются внешние API (Pexels, Pollinations AI, LoremFlickr).

**Рекомендация:** 
- Убедиться, что все изображения соответствуют политике контента
- Рассмотреть возможность хостинга изображений на собственном сервере
- Проверить лицензии на использование изображений

### 5. **Console.log в production**

**Проблема:** Есть несколько `console.log/warn/error` в коде.

**Решение:** 
- Удалить или обернуть в условие `if (process.env.NODE_ENV === 'development')`
- Или использовать более продвинутый логгер

### 6. **WebSocket соединения**

**Проблема:** Используется WebSocket для мультиплеера.

**Рекомендация:**
- Убедиться, что сервер доступен из домена Яндекс Игр
- Рассмотреть использование Яндекс Cloud Functions для бэкенда
- Или использовать Firebase/другой сервис, разрешенный платформой

## 🔧 Быстрые исправления:

### 1. Замена window.prompt и window.alert

Создать кастомные модальные окна в `client/index.html` и использовать их вместо нативных диалогов.

### 2. Удаление console.log

Обернуть все `console.*` вызовы:
```javascript
const DEBUG = window.location.hostname === 'localhost';
if (DEBUG) console.log(...);
```

### 3. Интеграция рекламы

Добавить в `client/main.js`:
```javascript
async function showAd() {
  if (platform.ysdk && platform.ysdk.adv) {
    try {
      await platform.ysdk.adv.showFullscreenAdv({
        callbacks: {
          onOpen: () => {},
          onClose: (wasShown) => {},
          onError: (error) => {},
        }
      });
    } catch (error) {
      console.warn('[ad] Failed to show ad:', error);
    }
  }
}
```

### 4. Интеграция лидербордов

Использовать `ysdk.getLeaderboards()` для отправки и получения данных лидерборда.
