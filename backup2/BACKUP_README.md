# Бэкап версии проекта

Это полный бэкап рабочей версии проекта "Нейросеть или Реальность" для Яндекс Игр.

**Дата создания бэкапа:** 13.11.2025

## Что включено в бэкап

- ✅ `client/` - вся клиентская часть (HTML, CSS, JS)
- ✅ `server/` - серверная часть (Express, WebSocket, данные)
- ✅ `scripts/` - вспомогательные скрипты
- ✅ `package.json` и `package-lock.json` - зависимости проекта
- ✅ `README.md` - основная документация
- ✅ `YANDEX_GAMES_COMPLIANCE.md` - соответствие требованиям Яндекс Игр
- ✅ `logo.png` - логотип проекта

## Как восстановить проект из бэкапа

### 1. Скопируйте содержимое папки backup2

Скопируйте все файлы из `backup2/` в нужную директорию (например, в корень нового проекта).

### 2. Установите зависимости

```bash
npm install
```

### 3. Запустите проект

```bash
npm run dev
```

Или отдельно:

```bash
# Сервер (порт 3000)
npm run dev:server

# Клиент (порт 4173)
npm run dev:client
```

## Структура проекта

```
backup2/
├── client/              # Клиентская часть
│   ├── index.html      # HTML структура
│   ├── main.js         # Основная логика игры
│   ├── styles.css      # Стили
│   └── logo.png        # Логотип
├── server/             # Серверная часть
│   ├── index.js        # Express сервер и WebSocket
│   └── data/           # Данные
│       ├── imageManifest.json  # База изображений
│       ├── quests.json         # Задания
│       └── bannedWords.json    # Запрещенные слова

├── package-lock.json   # Зафиксированные версии
├── README.md           # Основная документация

```
