# 🚀 Быстрый старт: Локальные AI-изображения

## ✅ Да, это работает!

**Важно:** Локально загружаются **только AI-фотографии**.  
Реальные фото продолжают загружаться с внешних сервисов (Pexels/LoremFlickr) автоматически.

Вы можете загрузить 1-2 тысячи AI-фотографий на сервер и использовать их в игре.

---

## 📝 3 простых шага

### 1️⃣ Загрузите изображения

Загрузите ваши AI-изображения в папку:
```
server/data/images/ai/
```

**Форматы:** `.jpg`, `.jpeg`, `.png`, `.webp`, `.gif`

### 2️⃣ Запустите скрипт

```bash
# Для localhost (разработка)
node scripts/add-local-images.mjs

# Для production (укажите ваш домен)
node scripts/add-local-images.mjs --api-base=https://api.your-domain.com
```

Скрипт автоматически:
- ✅ Найдет все изображения в папке
- ✅ Добавит их в `imageManifest.json`
- ✅ Создаст правильные URL для доступа

### 3️⃣ Перезапустите сервер

```bash
# Если используете PM2
pm2 restart game-server

# Или просто
npm run dev
```

**Готово!** Изображения теперь используются в игре.

---

## 📊 Пример

```bash
# 1. Загрузили 1000 изображений в server/data/images/ai/

# 2. Запустили скрипт
node scripts/add-local-images.mjs --api-base=https://api.your-game.com

# Вывод:
# ✅ Найдено изображений: 1000
# ✅ Добавлено новых: 1000
# ✅ Манифест сохранен

# 3. Перезапустили сервер
pm2 restart game-server
```

---

## 🔧 Дополнительные параметры

```bash
# Указать другую папку для AI-изображений
node scripts/add-local-images.mjs --dir=server/data/images/my-ai-photos

# Указать название источника
node scripts/add-local-images.mjs --source="My Custom AI Collection"
```

**Примечание:** Реальные фото не загружаются локально. Они автоматически обновляются из внешних источников при запуске `npm run update-manifest`.
