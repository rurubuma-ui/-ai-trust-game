# 📜 Информация о лицензиях изображений

## ⚠️ Важно понимать

### Google Custom Search API - это инструмент ПОИСКА, не источник изображений

1. **API бесплатный** ✅
   - 100 запросов/день бесплатно
   - Это только поиск изображений в интернете

2. **Изображения имеют разные лицензии** ⚠️
   - Google находит изображения по всему интернету
   - Каждое изображение принадлежит своему автору
   - Лицензии могут быть разными

3. **Фильтр Creative Commons** 🔍
   - Мы используем фильтр `rights=cc_...` для поиска изображений с CC лицензиями
   - Это помогает найти изображения с открытыми лицензиями
   - **НО:** фильтр не гарантирует 100% точность

## 📋 Типы Creative Commons лицензий

- **CC0 (Public Domain)** - можно использовать свободно, без указания автора
- **CC BY** - можно использовать, но нужно указать автора
- **CC BY-SA** - можно использовать с указанием автора, производные работы должны иметь ту же лицензию
- **CC BY-NC** - можно использовать только в некоммерческих целях
- **CC BY-ND** - можно использовать, но нельзя изменять

## ✅ Что мы делаем правильно

1. Используем фильтр Creative Commons при поиске
2. Сохраняем информацию об источнике в манифесте
3. Сохраняем ссылку на оригинальный источник (`originalUrl`)

## ⚠️ Что нужно учитывать

1. **Проверка лицензий** - рекомендуется проверять лицензию каждого изображения на исходном сайте
2. **Указание автора** - для некоторых CC лицензий требуется указание автора
3. **Коммерческое использование** - некоторые лицензии запрещают коммерческое использование

## 🎯 Рекомендации для вашего проекта

### Для игры на Yandex Games:

1. **Используйте изображения с CC0 или CC BY** - самые свободные лицензии
2. **Сохраняйте информацию об авторах** - на случай, если понадобится указать
3. **Проверяйте лицензии** - особенно для коммерческих проектов

### Альтернативные источники с гарантированными лицензиями:

1. **Pexels** - все изображения с лицензией Pexels (можно использовать свободно)
2. **Unsplash** - все изображения с Unsplash License (можно использовать свободно)
3. **Pixabay** - все изображения с Pixabay License (можно использовать свободно)

## 📝 Текущая реализация

В коде мы:
- Сохраняем `source.license: 'Creative Commons (verify on source site)'`
- Сохраняем `source.originalUrl` для проверки лицензии
- Используем фильтр CC при поиске через Google
