# 🎯 Краткая сводка: Как всё работает

## 📋 В двух словах

1. **Яндекс Игры** хостят только клиентскую часть (HTML/CSS/JS)
2. **Ваш backend сервер** (отдельно) обрабатывает логику игры
3. **Клиент** подключается к backend через HTTPS/WebSocket
4. **Backend** генерирует вопросы и управляет матчами
5. **Изображения** загружаются с внешних источников (Pexels, Pollinations и т.д.)

---

## 🔄 Поток данных Single Player

```
Пользователь → Яндекс Игры → Загружает JS
                                   ↓
                            JS выполняется в браузере
                                   ↓
                            HTTP запрос → Ваш backend
                                   ↓
                            Backend создает вопрос
                                   ↓
                            Возвращает JSON с URL изображения
                                   ↓
                            Клиент загружает изображение
                                   ↓
                            Пользователь отвечает
                                   ↓
                            HTTP запрос с ответом → Backend
                                   ↓
                            Backend проверяет ответ
                                   ↓
                            Возвращает результат + следующий вопрос
```

---

## 🎮 Поток данных Multiplayer

```
Игрок 1 → Яндекс Игры → WebSocket → Ваш backend
                                           ↓
                                    Создается match
                                           ↓
Игрок 2 → Яндекс Игры → WebSocket → Ваш backend
                                           ↓
                                    Оба игрока получают вопросы
                                           ↓
                                    Оба отвечают
                                           ↓
                                    Backend определяет победителя раунда
                                           ↓
                                    Обновляется счет
                                           ↓
                                    Повторяется до 10 очков
```

## 🏗️ Архитектура

```
┌─────────────────┐
│  Яндекс Игры    │  ← Хостит только статику
│  (CDN)          │
└────────┬────────┘
         │ HTTPS/WSS
         ↓
┌─────────────────┐
│  Ваш Backend    │  ← Обрабатывает логику
│  (Ваш сервер)   │
└─────────────────┘
```

## 📝 Что нужно для production

1. ✅ Backend сервер (VPS/Cloud)
2. ✅ Домен для API (например: `api.your-game.com`)
3. ✅ SSL сертификат (Let's Encrypt - бесплатно)
4. ✅ Обновить `API_BASE` в `client/main.js`
5. ✅ Настроить CORS на backend (уже сделано)
6. ✅ Задеплоить клиент на Яндекс Игры
